# 虚拟账户体系


### 模块定义

虚拟账户系统是整个资金体系的“账本层（Ledger Layer）”。

### 设计目标

- 将资金结构数字化
- 支持多层账户体系
- 实现实时对账
- 支持多币种统一管理

### 账户结构

支持多层级结构：
- Platform Account（平台账户）
- Merchant Account（商户账户）
- User Account（用户账户）

### 核心能力

- 多币种余额系统
- 实时账本更新（Real-time Ledger）
- 自动对账（Auto Reconciliation）
- 账户间转账（Internal Transfer）
- API 创建与管理账户

### 示例（资金流）

- 用户充值 → User VA
- 商户收入 → Merchant VA
- 平台手续费 → Platform VA
所有流动均记录在统一账本中
